快速幂、逆元与线性同余方程

一、快速幂

1.1 问题背景

常规计算 abmod  pa^b \mod pabmodp 的方式是循环乘 bbb 次，时间复杂度为 O(b)O(b)O(b) 。当 bbb 达到 101810^{18}1018 时，该方法完全不可行。

快速幂通过二进制分解指数，将时间复杂度降至O(log⁡b)O(\log b)O(logb) ，是处理大指数幂运算的核心算法。

定理

三、逆元的定义与存在性

2.1 逆元定义

对于整数 aaa 和模数 ppp ，若存在整数 xxx 满足 a×x≡1(modp)a \times x \equiv 1 \pmod{p}a×x≡1(modp) ，则称 xxx 为 aaa 在模 ppp 下的乘法逆元，记为 a−1mod  pa^{-1} \mod pa−1modp 。

2.2 逆元存在的充要条件aaa 在模 ppp 下存在逆元的充要条件是 gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1 （即 aaa 与 ppp 互质）。

证明

  • 必要性：若 aaa 存在逆元 xxx ，则 ax−1=kpa x - 1 = k pax−1=kp （ k∈Zk \in \mathbb{Z}k∈Z ），即 ax−kp=1a x - k p = 1ax−kp=1 。根据贝祖定理， gcd⁡(a,p)\gcd(a, p)gcd(a,p) 必整除 111，故 gcd(a,p)=1gcd(a, p) = 1gcd(a,p)=1。
  • 充分性：若 gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1 ，由贝祖定理，存在整数 x,yx, yx,y 使得 ax+py=1a x + p y = 1ax+py=1 。两边模 ppp 得 ax≡1(modp)a x \equiv 1 \pmod{p}ax≡1(modp) ，即 xxx 是 aaa 的逆元。

4.1 问题背景

欧几里得算法仅能求 gcd⁡(a,b)\gcd(a, b)gcd(a,b) ，但扩展欧几里得算法（EXGCD）可在求 gcd⁡(a,b)\gcd(a, b)gcd(a,b) 的同时，找到整数 x,yx, yx,y 满足贝祖等式：ax+by=gcd⁡(a,b)a x + b y = \gcd(a, b)ax+by=gcd(a,b)。

核心思路

  1. 边界条件：当 b=0b = 0b=0 时， gcd⁡(a,0)=a\gcd(a, 0) = agcd(a,0)=a ，此时贝祖等式为 a×1+0×0=aa \times 1 + 0 \times 0 = aa×1+0×0=a ，即 x=1,y=0x=1, y=0x=1,y=0 。

  2. 递归步骤：假设已求得 gcd⁡(b,amod  b)\gcd(b, a \mod b)gcd(b,amodb) 对应的解 x′,y′x', y'x′,y′ ，即：

    bx′+(amod  b)y′=gcd⁡(a,b)b x' + (a \mod b) y' = \gcd(a, b)bx′+(amodb)y′=gcd(a,b)

    由于 amod  b=a−⌊ab⌋×ba \mod b = a - \lfloor \frac{a}{b} \rfloor \times bamodb=a−⌊ba​⌋×b ，代入上式：

    bx′+(a−⌊ab⌋×b)y′=gcd⁡(a,b)b x' + (a - \lfloor \frac{a}{b} \rfloor \times b) y' = \gcd(a, b)bx′+(a−⌊ba​⌋×b)y′=gcd(a,b)

    整理得：

    ay′+b(x′−⌊ab⌋×y′)=gcd⁡(a,b)a y' + b (x' - \lfloor \frac{a}{b} \rfloor \times y') = \gcd(a, b)ay′+b(x′−⌊ba​⌋×y′)=gcd(a,b)

    x=y′,y=x′−⌊ab⌋×y′x = y', \quad y = x' - \lfloor \frac{a}{b} \rfloor \times y'x=y′,y=x′−⌊ba​⌋×y′

证明（数学归纳法）

  • 基例： b=0b=0b=0 时，解为 x=1,y=0x=1, y=0x=1,y=0 ，显然满足 a×1+0×0=a=gcd⁡(a,0)a \times 1 + 0 \times 0 = a = \gcd(a, 0)a×1+0×0=a=gcd(a,0) 。
