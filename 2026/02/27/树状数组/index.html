树状数组（Binary Indexed Tree）

树状数组（也称为Fenwick Tree）是一种高效的前缀和查询与单点更新的数据结构，相较于线段树，它实现更简单、空间复杂度更低，能在 O(log n) 的时间复杂度内完成单点修改和前缀和查询操作，是竞赛和工程中处理区间问题的常用工具。

围绕树状数组的核心原理、实现及应用展开，包含lowbit运算、基础操作、经典例题（逆序对、区间更新、二分查询）和高维树状数组拓展，所有例题均适配 105 ∼ 2 × 105 级别的数据范围，符合算法竞赛的实际要求。

主讲知识点框架

  1. lowbit运算：树状数组的核心数学基础
  2. 树状数组基础：定义、单点修改、前缀和查询（例题1）
  3. 经典应用：逆序对统计（例题2）、区间更新+区间查询（例题3）
  4. 进阶操作：树状数组上的二分查询（例题4）
  5. 拓展内容：高维树状数组（二维，例题5）

一、核心基础：lowbit运算

1.1 定义

lowbit(x)表示非负整数x的二进制表示中，最低位的1所对应的值。

例如：

  • x = 6 （二进制110），lowbit(x)=2（二进制10）
  • x = 8 （二进制1000），lowbit(x)=8（二进制1000）
  • x = 5 （二进制101），lowbit(x)=1（二进制1）

1.2 计算公式

在计算机中，利用补码特性可通过位运算直接计算：

lowbit(x) = x&(−x)

原理：计算机中负数以补码形式存储，-x = ~x + 1（按位取反+1）。对x和-x做按位与运算时，仅最低位的 1 会被保留，其余位全部置0，恰好得到lowbit值。

1.4 编程技巧拓展

lowbit 不仅用于树状数组，也是竞赛中的常用位运算技巧：

  1. 状态压缩中，快速枚举一个二进制数的所有子集；
  2. 快速判断一个数是否为2的整数次幂：lowbit(x) == x；
  3. 快速消除一个数的最低位1：x -= lowbit(x)，可用于统计二进制中1的个数。

二、树状数组基础定义与操作

2.1 数组定义

树状数组用一个一维数组c[]模拟树形结构，每个节点 c[x] 管理一段长度为 lowbit(x) 的连续区间，维护这段区间的区间和。

c[]（通常称为树组），满足父子节点关系：

  • 节点i的父节点为i + lowbit(i)
  • 节点i的子节点为i - lowbit(i)
  • 树状数组的下标从1开始（避免lowbit(0)无意义）。

2.2 两个核心操作

树状数组的核心能力是单点修改和前缀和查询，时间复杂度均为 O(log n) 。

操作1：单点修改 modify(x, s)

功能：将原数组a[x]的值增加s（若需赋值为d，可计算s = d - a[x]后调用）。

原理：从x出发，不断向上更新父节点的树组值，直到超出数组长度n。

三、树状数组经典应用

3.1 例题2：逆序对2（树状数组统计逆序对）

题目描述

给定n个互不相同的数，求逆序对的数量（逆序对： i < j 且 a[i] > a[j] ）。

解题思路

逆序对统计是树状数组的经典应用，核心思路是从后往前遍历，利用树状数组统计已遍历元素中比当前元素小的数的个数，累加即为逆序对总数；

若数据范围不连续，需先做离散化（本题已保证 1 ≤ a[i] ≤ n ，可直接使用）。

优化技巧：将a[i]映射为n+1-a[i]，将“找比当前小的数”转化为“找比当前大的数”，简化统计逻辑。

核心步骤

  1. 初始化空的树状数组；

  2. 从i=n到i=1遍历原数组：

    • 逆序对数量 += query(a[i])（查询已插入的、比当前a[i]小的数的个数）；
    • 调用modify(a[i], 1)（将当前数插入树状数组，计数+1）。

3.2 例题3：树状数组2（区间更新+区间查询）

题目描述

给定n个初始为0的数，支持q个操作：

  1. 1 l r d：将区间a[l~r]的每个数增加d；
  2. 2 x：查询a[1~x]的前缀和，结果对 264 取模（即使用无符号64位整数）。

解题思路

树状数组本身仅支持单点修改，需通过差分思想将区间更新转化为两次单点修改，再结合数学推导实现区间查询。

核心推导

设原数组为a[]，差分数组为d[]（ d[1] = a[1], d[i] = a[i] − a[i − 1] ），则原数组的前缀和为：

$\sum_{i=1}^x a[i] = \sum_{i=1}^x \sum_{j=1}^i d[j] = \sum_{i=1}^x d[i] \times (x+1 - i) = (x+1)\sum_{i=1}^x d[i] - \sum_{i=1}^x d[i] \times i$

因此，维护两个树状数组：

  • c1[]：存储差分数组d[]的前缀和；
  • c2[]：存储d[i] * i的前缀和。

则前缀和查询公式为：

query_sum(x) = (x + 1) × c1.query(x) − c2.query(x)

区间更新转化

对[l, r]增加d，等价于对差分数组做：

  • d[l] += d；
  • d[r+1] -= d（若r+1 <=n）。

对应两个树状数组的修改：

  • c1.modify(l, d)、c1.modify(r+1, -d)；
  • c2.modify(l, l*d)、c2.modify(r+1, -(r+1)*d)。

关键注意：使用unsigned long long（u64）自动实现 264 取模，避免手动处理溢出；用模板类封装树状数组，提升代码复用性。

四、树状数组进阶：树上二分查询

4.1 例题4：树状数组二分（前缀和二分找最大T）

题目描述

给定n个数，支持q个操作：

  1. 1 x d：将a[x]赋值为d；
  2. 2 s：查询最大的T（ 0 ≤ T ≤ n ），满足 $\sum_{i=1}^T a[i] \le s$ 。

解题思路

树状数组的前缀和满足单调性（原数组元素均为正），因此可在树状数组上直接二分，找到满足条件的最大T，时间复杂度 O(log2n) 。

二分原理

从最高位（如18位，覆盖 218 = 262144 > 2 × 105 ）向最低位遍历，尝试将当前位加入答案pos，若加入后前缀和仍  ≤ s ，则保留该位，否则跳过。

五、树状数组拓展：高维树状数组

5.1 二维树状数组（例题5）

题目描述

给定 n × m 的二维数组，支持q个操作：

  1. 1 x y d：将a[x][y]赋值为d；
  2. 2 x y：查询左上角 (1, 1) 到右下角 (x, y) 的二维前缀和。

解题思路

二维树状数组是一维树状数组的直接拓展，核心是两层循环实现单点修改和二维前缀和查询，时间复杂度 O(log n × log m) 。

数组定义

用二维数组c[N][M]存储二维树状数组，下标均从1开始。

核心操作
  1. 单点修改：对 (x, y) 增加s，两层循环分别更新x和y的父节点；
  2. 二维前缀和查询：查询 (1, 1) 到 (x, y) 的和，两层循环分别累加x和y的子节点。

操作实现说明

  • 赋值操作：modify(x, y, d - a[x][y])，再更新a[x][y] = d；
  • 二维前缀和查询：直接调用query(x, y)。

注：高维树状数组的空间复杂度随维度指数增长，因此通常仅使用二维，更高维度（如三维）一般用线段树或其他方法替代。

六、树状数组总结

6.1 核心特性

  1. 时间复杂度：单点修改、前缀和查询均为 O(log n) ，二维为 O(log nlog m) ；
  2. 空间复杂度： O(n) （一维）、 O(nm) （二维），远优于线段树；
  3. 局限性：仅支持前缀和相关操作，无法直接处理任意区间的最值查询（最值需用线段树）。

6.2 适用场景

  1. 单点修改 + 前缀和/区间和查询；
  2. 区间修改 + 前缀和/区间和查询（差分+双树状数组）；
  3. 逆序对、第K小数、数频统计等计数问题；
  4. 二维单点修改 + 二维前缀和查询（n,m较小的场景）。

6.3 竞赛技巧

  1. 下标从0开始的死循环：树状数组必须从1开始，lowbit(0)=0会导致循环无法终止；
  2. 数据溢出：前缀和极易超过int范围，必须用long long存储，mod 2^64用unsigned long long；
  3. 数组大小不足：区间修改时会访问到r+1，树状数组大小需开至n+10，避免越界；
  4. 离散化错误：离散化时必须先排序、去重，再用lower_bound映射，保证相对大小不变；
  5. 多组数据未清空：多组测试用例时，必须手动清空树状数组c[]和原数组a[]，否则会残留上一组数据；
  6. 功能局限性：树状数组仅能维护满足结合律、可差分的信息（和、异或、前缀最值），无法处理复杂的区间修改（如区间乘、区间赋值），这类场景需用线段树。

6.4 与线段树的对比

数据结构 实现难度 空间复杂度 单点修改 区间查询 最值查询
树状数组 简单 O(n) O(log n) O(log n) 不支持
线段树 复杂 O(4n) O(log n) O(log n) 支持

结论：若问题仅涉及和相关的单点/区间操作，优先使用树状数组；若涉及最值、区间覆盖等操作，使用线段树。

七、课后练习

  1. 基础练习：实现树状数组的区间和查询（利用前缀和相减：query(r) - query(l-1)）；
  2. 进阶练习：用树状数组解决带修逆序对问题（支持单点修改+查询当前逆序对数量）；
  3. 拓展练习：实现二维树状数组的区间更新+区间查询（结合二维差分）。
