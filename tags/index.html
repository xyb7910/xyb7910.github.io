🏷️ 算法标签体系

为了方便你快速找到对应题型的题解，我按「算法知识点 + 难度 + 刷题平台」整理了标签，点击标签即可查看所有相关题解：

📌 标签分类说明

标签类型 示例标签 说明
核心知识点 数组、链表、二叉树、动态规划、贪心 按算法考点划分（核心）
刷题平台 LeetCode、AcWing、牛客网 按题源划分
难度等级 简单、中等、困难 按题目难度划分
解题技巧 双指针、滑动窗口、递归、分治 按解题方法划分
面试高频 大厂面试题、校招真题 针对性面试题

💡 标签使用技巧

  1. 组合筛选：比如想找「LeetCode + 中等 + 动态规划」的题，先点「动态规划」再在结果里筛选；
  2. 新手推荐：优先看「简单 + 数组/链表」标签，打好基础；
  3. 面试冲刺：直接看「大厂面试题」标签，聚焦高频考点。

标签会随刷题进度持续更新，若你想找某类未覆盖的标签，可留言告诉我～
